Running Locally?

You're running this page directly from your file system, which can cause CORS errors. Here are ways to solve this:

Option 1: Use a CORS proxy

CORS Anywhere CORS Proxy IO All Origins

Option 2: Use a browser extension

CORS Unblock Extension

Option 3: Use a local web server

Open a terminal, navigate to this folder and run: python -m http.server then visit http://localhost:8000
